############ IONIC DOCS
Version Ionic: v4 -> https://ionicframework.com/docs/v4/api/toast

############ CAPACITOR
ionic integrations enable capacitor
npm i @capacitor/core
npm i -D @capacitor/cli
npm install @capacitor/preferences

############ PUBLISH ANDROID
ionic build (cria pasta www)
ionic cap add android (cria pasta android) 
ionic cap copy
ionic cap open android (chama no ANDROID STUDIO)

Obs: Se houve alteração no codigo e quer refletir no emulador, pode executar o comando abaixo, pois ele é mais rapido:
ionic cap copy android

Doc: https://ionicframework.com/docs/angular/your-first-app/deploying-mobile

TO ONLY RUN in MOBILE: 
ionic serve --external (open IP in phone)
